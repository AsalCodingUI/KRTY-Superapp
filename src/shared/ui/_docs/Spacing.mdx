# Spacing & Layout

We use a standard 4px grid system.

## Spacing Scale

| Token | Rem  | Px   | Usage                    |
| ----- | ---- | ---- | ------------------------ |
| `0`   | 0    | 0px  | -                        |
| `1`   | 0.25 | 4px  | Tight spacing, icon gaps |
| `2`   | 0.5  | 8px  | Small component padding  |
| `3`   | 0.75 | 12px | Medium component padding |
| `4`   | 1    | 16px | Standard layout gap      |
| `5`   | 1.25 | 20px | -                        |
| `6`   | 1.5  | 24px | Section padding          |
| `8`   | 2    | 32px | Large section padding    |
| `10`  | 2.5  | 40px | -                        |
| `12`  | 3    | 48px | -                        |
| `16`  | 4    | 64px | Page margins             |
| `20`  | 5    | 80px | -                        |
| `24`  | 6    | 96px | Hero sections            |

## Border Radius

Consistency in roundness makes the UI feel cohesive.

| Token  | Size   | Usage                            |
| ------ | ------ | -------------------------------- |
| `none` | 0px    | Square elements                  |
| `sm`   | 4px    | Small inputs, badges, checkboxes |
| `md`   | 8px    | Buttons, inputs, standard cards  |
| `lg`   | 12px   | Modals, large cards              |
| `xl`   | 16px   | Large containers                 |
| `2xl`  | 24px   | Drawers, very large containers   |
| `full` | 9999px | Pills, avatars, circular buttons |

## Elevation (Shadows)

| Token        | Usage                          |
| ------------ | ------------------------------ |
| `shadow-xs`  | Subtle borders, inset elements |
| `shadow-sm`  | Interactive elements (hover)   |
| `shadow-md`  | Dropdowns, popovers, cards     |
| `shadow-lg`  | Modals, sticky headers         |
| `shadow-xl`  | Large dialogs                  |
| `shadow-2xl` | Floating interactions          |
